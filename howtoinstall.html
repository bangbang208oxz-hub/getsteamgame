How to Install
วิธีติดตั้ง

Follow these steps to install games on Steam.
ทำตามขั้นตอนเหล่านี้เพื่อติดตั้งเกมบน Steam

🇺🇸 | How to Install Games English
1
Download Program
Download the program to install the game on the Steam app.
2
Disable Your Antivirus
Turn off real-time protection before continuing.
3
Run the Application
Launch the program included in the ZIP file and follow the on-screen instructions until the process is complete.
4
Enter the App ID & Game ID
Enter the App ID and Game ID of the game you want to install.
5
Wait for the Process to Complete
Please wait until the setup process finishes automatically.
6
Return to the Steam App
Go back to Steam and start the game installation.
7
Re-enable Your Antivirus
Turn real-time protection back on as before.
🇹🇭 | วิธีติดตั้งเกม ภาษาไทย
1
ดาวน์โหลดโปรแกรม
ดาวน์โหลดโปรแกรมเพื่อติดตั้งเกมบนแอพ Steam
2
ปิดแอนตี้ไวรัส
ปิด real-time protection ก่อนดำเนินการต่อ
3
รันโปรแกรม
ทำตามขั้นตอนการติดตั้งจนเสร็จสมบูรณ์
4
กรอก App ID & ID GAME
กรอกข้อมูล App ID และ ID GAME ของเกมที่จะติดตั้ง
5
รอให้โปรแกรมดำเนินการเสร็จสิ้น
โปรดรอจนกว่าการตั้งค่าจะเสร็จสมบูรณ์
6
กลับไปที่แอพ Steam
จากนั้นกดติดตั้งเกมได้เลย
7
เปิดแอนตี้ไวรัส
เปิด Real-time Protection ให้กลับเป็นเหมือนเดิม
